This is a test about the approximation of a curve via a spline.

Either open it via the "Binder" link below:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/pthom/FitSpline/HEAD)

Or install a python virtual environment like this, and then run jupyter notebook (or open Visual Studio Code)

````
python3 -m pip install virtualenv
python3 -m virtualenv venv
source venv/bin/activate
pip install numpy opencv-python matplotlib scipy jupyter
````
